# Essentia Python tutorial
A tutorial for using Essentia in Python

How to run:

```
git clone https://github.com/MTG/essentia-tutorial.git
docker pull mtgupf/mir-toolbox
cd essentia-tutorial
./run.sh
```

Go to http://localhost:8888/

Password: `mir`
